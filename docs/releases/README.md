# Uya Release Notes

> 历史路径说明：本目录中的早期 release notes 可能包含 `compiler-c/`、`compiler-mini/`、`bin/uya-c`、`make tests-c` 等旧路径。这些历史路径仅用于版本记录，不代表当前开发入口；当前入口以仓库根目录 `Makefile`、`docs/DEVELOPMENT.md` 和 `AGENTS.md` 为准。

新增 release notes 时，优先记录当前 `src/` 自举编译器、`make from-c` / `make from-c-native`、`make uya` / `make uya-hosted`、`make check-core` / `make check` 和 `make release-clean` 等现行流程。
